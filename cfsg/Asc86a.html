FINITE GROUP THEORY, Second Edition

M. ASCHBACHER

An algebraic structure is a set with defined operations on its elements.

An algebraic structure with one operation, which acts on two elements, is said to be a group if it satisfies the below conditions:
The operation is restricted to the set.
There exists an element in the set that acts as a global identity.
For every element in the set, there is an element in the set that acts as its inverse.
The operation is associative.

The order of a group G is an integer, and is denoted |G|. It is the cardinality of its underlying set.

G is said to be a finite group if its order is finite.

G is said to be an abelian group if every element commutes with every other element.
